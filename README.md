# Moonshot Fullstack Engine 🚀

A high-performance MERN stack application featuring secure authentication and project management.

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS, Axios, Lucide Icons
- **Backend:** Node.js, Express, MongoDB (Mongoose)
- **Security:** JWT (JSON Web Tokens), Bcrypt Hashing, Zod Validation

## Setup

1. `cd server && npm install`
2. `cd client && npm install`
3. Create a `.env` in `/server` with `PORT`, `MONGODB_URI`, and `JWT_SECRET`.
4. Run server: `node index.js`
5. Run client: `npm run dev`
